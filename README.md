# Blabber
## Conversation to comic generator

Powered by [blabber-comic](https://github.com/sharpshark28/blabber-comic).

## Sample

![Preview](/comics/example.png)

## Installing

`npm install -g blabber`

## Using

```
$ blabber
$ Who?: Glen
$ Said What?: Test 1
$ Another? (y/n) y
$ Who?: Sarah
$ Said What?: Test 2
$ Another? (y/n) y
$ Who?: Glen
$ Said What?: Test 3
$ Another? (y/n) n
$ Generating Comic...
$ [ Comic Generated @ comics/1.png ]
```
